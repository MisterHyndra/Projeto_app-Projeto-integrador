Guia de Contribuição

Obrigado pelo seu interesse em contribuir para o Med Alert! Este guia irá ajudá-lo a configurar o ambiente de desenvolvimento e enviar suas contribuições.

Sumário

  • Primeiros Passos
  • Fluxo de Trabalho
  • Padrões de Código
  • Enviando Mudanças

Primeiros Passos

1. Fork do Projeto

Comece fazendo um fork do repositório principal para a sua conta do GitHub.

2. Clone o Repositório

Clone o seu fork para sua máquina local:

# Clone seu fork
git clone https://github.com/seu-usuario/med-alert.git

# Navegue até o diretório do projeto
cd med-alert

# Adicione o repositório original como remoto
git remote add upstream https://github.com/original/med-alert.git

3. Configure o Ambiente

Siga as instruções no Guia de Instalação para configurar o ambiente de desenvolvimento.

Fluxo de Trabalho

1. Mantenha seu Fork Atualizado

# Busque as mudanças do repositório original
git fetch upstream

# Atualize sua branch main
git checkout main
git merge upstream/main

2. Crie uma Branch para sua Feature

# Crie e mude para uma nova branch
git checkout -b feature/nome-da-feature

# Para correções de bug
git checkout -b fix/descricao-do-bug

3. Faça Commits Atômicos

Faça commits pequenos e frequentes com mensagens claras:

git add .
git commit -m "feat: adiciona funcionalidade X"

4. Envie suas Alterações

git push -u origin sua-branch

Padrões de Código

JavaScript/TypeScript

  • Siga o JavaScript Standard Style
  • Use const por padrão, let apenas quando necessário
  • Prefira funções de seta

React/React Native

  • Use componentes funcionais com Hooks
  • Nomeie componentes com PascalCase
  • Use prop-types ou TypeScript

Enviando Mudanças

1. Crie um Pull Request

  1. Vá para a página do seu fork no GitHub
  2. Clique em "Compare & pull request"
  3. Preencha o formulário do PR
  4. Marque a issue relacionada (ex: "Fixes #123")

2. Revisão de Código

Após abrir o PR, a equipe revisará suas alterações. Esteja preparado para fazer ajustes se necessário.

Obrigado!

Agradecemos sua contribuição para o Med Alert. Juntos, podemos ajudar as pessoas a gerenciarem melhor seus medicamentos.
